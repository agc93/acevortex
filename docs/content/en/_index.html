{{< /blocks/cover >}} {{% blocks/lead color="primary" %}} AceVortex is a simple extension to add support for Ace Combat 7 Skies Unknown to the Vortex mod manager. With this extension you can install, remove and manage your AC7 mods using Vortex, perfect for players already familiar with Vortex. **This project is unofficial!** We have no association with Nexus Mods nor with Project Aces and is only a community project. {{% /blocks/lead %}} {{< blocks/section color="dark" >}} {{% blocks/feature icon="fa-tools" title="Full Vortex integration" %}} Manage your Ace Combat mods the same way you manage any other Vortex-supported game. {{% /blocks/feature %}} {{% blocks/feature icon="fab fa-github" title="Contributions welcome!" url="https://github.com/agc93/acevortex" %}} AceVortex is fully open-source and open to community contributions and fixes! {{% /blocks/feature %}} {{% blocks/feature icon="fas fa-cloud-download-alt" title="Supports Most Mods" %}} You should be able to install almost any mod for AC7, now with improved skin installation support. {{% /blocks/feature %}} {{< /blocks/section >}} {{< blocks/section >}}
