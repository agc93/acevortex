AceVortex: Ace Combat 7 Support for Vortex

Documentation GitHub Nexus Mods

Adds support for Ace Combat 7 Skies Unknown mods to the Vortex mod manager. Fully open-source.

AceVortex is a simple extension to add support for Ace Combat 7 Skies Unknown to the Vortex mod manager. With this extension you can install, remove and manage your AC7 mods using Vortex, perfect for players already familiar with Vortex.

This project is unofficial! We have no association with Nexus Mods nor with Project Aces and is only a community project.

Full Vortex integration

Manage your Ace Combat mods the same way you manage any other Vortex-supported game.

Contributions welcome!

AceVortex is fully open-source and open to community contributions and fixes!
